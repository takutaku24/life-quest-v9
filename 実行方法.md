# 🚀 ローカルでの実行方法

## Windowsでの実行手順

### 1. ターミナル（PowerShell）を開く

- **方法A**: Windowsキーを押して「PowerShell」と検索 → 「Windows PowerShell」を開く
- **方法B**: エクスプローラーで `c:\Users\takut\OneDrive\Desktop\life_quest` を開いて、アドレスバーに `powershell` と入力してEnter

### 2. プロジェクトフォルダに移動

```powershell
cd c:\Users\takut\OneDrive\Desktop\life_quest
```

### 3. 必要なパッケージをインストール（初回のみ）

```powershell
pip install -r requirements.txt
```

### 4. アプリを起動

```powershell
streamlit run app.py
```

### 5. ブラウザで開く

コマンド実行後、自動的にブラウザが開きます。
開かない場合は、ターミナルに表示されるURL（例: `http://localhost:8501`）をブラウザで開いてください。

## よくある問題

### 「streamlit が見つかりません」エラーが出る場合

```powershell
pip install streamlit
```

### 「gspread が見つかりません」エラーが出る場合

```powershell
pip install gspread oauth2client pandas altair
```

### ポート8501が既に使われている場合

別のポートで起動：
```powershell
streamlit run app.py --server.port 8502
```

### アプリを停止する場合

ターミナルで `Ctrl + C` を押してください。
